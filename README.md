# random_network_maker
- 원본 그래프를 입력받아 그래프 특성을 사용해 무작위 그래프 (ER, Configuration, Chung-lu, BA)을 만들려고 한다.

##클래스 설명
1. random_graph_list
- 만들 수 있는 무작위 그래프 이름을 나열한다. 리스트에서 정의된 이름을 넣어야 그래프를 만들 수 있다.

2. ER 그래프를 만드는 함수
- ER 그래프란 ?\
 -> 원본 그래프의 노드 수를 동일하게 하고, 균일 연결 확률 p를 이용해 노드를 무작위로 연결해놓은 그래프이다.
  1) set_init_ER
  - ER 그래프를 만들기 위한 엣지 연결 확률을 만드는 데, 사용하는 함수이다.
  - 평균 차수와 노드수를 이용해 엣지 연결 확률을 계산한다. 이 때, 소수 2번째 자리까지 반환하도록 한다.
  - 엣지 연결 확률 계산 
  - 예외
    - (1) 노드 수가 0 일 때와 (2) 엣지 수가 0 일 때는 연결 확률을 계산할 수 없도록 한다.
  
  2) create_ERnp_graph
  - 원본 그래프의 노드와 set_init_ER에서 만든 연결확률을 이용해 ER 그래프를 만들도록 한다.
  - 예외
    - 연결확률 p가 0과 1 사이의 값이 아닌 경유 해당 그래프를 만들 수 없도록 한다.
